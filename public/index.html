Jipi Broker API

Welcome to the Jipi Broker API service. This service provides episode resolution for anime content.

API Endpoints

GET /api/resolve

Resolve episode URLs or direct URLs to streaming links.

Parameters:

  • episode - Episode ID in format: imdb_id:season:episode
  • url - Direct URL to resolve

Example:

GET /api/resolve?episode=tt0388629:1:1

Getting Started

To use this API, make GET requests to the /api/resolve endpoint with either an episode or url parameter.
